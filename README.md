# Decentralized Carbon Credit Marketplace

A decentralized platform for verifying, trading, and managing carbon credits. The system enables seamless connections between carbon offset projects and buyers, ensuring transparency and trust through AI-driven verification and blockchain ledger tracking.

## Features

- **Project Verification:** Automated AI scanning of satellite imagery and audit documents to verify carbon credit claims.
- **Credit Trading:** Buy and retire carbon credits directly through the platform.
- **Ledger Tracking:** Transparent tracking of all transactions and credit retirements.
- **Multi-Role Support:** Dedicated dashboards for both Buyers and Sellers.

## Tech Stack

- **Backend:** Node.js (with optional FastAPI Python implementation)
- **Frontend:** HTML, Vanilla CSS, and JavaScript
- **AI Integration:** OpenAI, Grok, and Google Gemini support for document verification
